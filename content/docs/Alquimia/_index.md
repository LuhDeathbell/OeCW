# Alquimia

A ação de criar poções não é mera ciência ou ato arcano, mas uma arte em seu próprio direito.

O efeito de uma poção pode ser calculado utilizando o cálculo abaixo:

    > (efeitoBase*solvente)+(alquimia*1.5*raridade)
